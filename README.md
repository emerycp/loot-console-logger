# ![icon](https://user-images.githubusercontent.com/38258431/223797326-aa318a83-6882-48ee-9c40-ea0e012ce98f.png) Loot Console Logger ![image](https://img.shields.io/github/v/release/emerycp/loot-console-logger?include_prereleases&logo=github)

Loot Console Logger is a RuneLite plugin that keeps your loot easy to follow in a texted based feed.

It can send drops to the in-game chatbox, track your session in a side panel, and help you filter out junk while keeping important drops easy to spot. It is especially useful if you play on a smaller screen or juggle multiple accounts at once.

## What it does

- Logs drops to the in-game message box
- Tracks your current session in a sidebar panel
- Shows total GE and HA value for the session
- Groups loot by source so you can quickly see what each monster or boss dropped
- Keeps a recent drops list with highlighted items marked
- Lets you ignore monsters you do not care about
- Lets you ignore or highlight specific items you want to notice or hide

## Filtering and highlighting

You can combine manual rules with automatic value or quantity rules.

Examples:
- Highlight `Hammer`
- Highlight anything worth at least `100` GE
- Ignore `Bones`
- Ignore anything worth less than `100` GE
- Highlight large rune drops without listing every rune by name

## Rule editing

The sidebar includes built-in rule editors for:
- Ignored Items
- Highlighted Items
- Ignored Monsters

Item rules have search suggestions based on in-game items, so you can pick from a list instead of typing everything by hand. Monsters' names doesn't support suggestions yet.

You can add them in the plugin settings separated by commas if you prefer.

## Session panel

The side panel is meant to be quick to read while you play.

It includes:

- Session totals
- Source summaries
- Recent drops
- Removable cards for drops and sources
- Searchable rule tags

You can also sort top items by:
- Count
- Total Value (GE)
- Total Value (HA)

- Logging to chat can be turned on or off in the plugin settings
